Pythonの算術演算子

English Vietnamese

20 1月 ,2021

Pythonの算術演算子について説明します。この記事を通じてPythonでの加算や減算、乗算、除算など四則演算を含め、算術演算子の使い方を学びましょう。

Pythonの算術演算子

Pythonの算術演算子は、次の9つの基本的な形式で表されます。これらの演算子は、整数、実数、複素数など、Pythonのすべての数値データ型に適用できます。

算術演算子書き方例意味
加算x + y20 + 3 = 23足し算
減算x - y20 - 3 = 17引き算
乗算x * y20 * 3 = 60掛け算
除算x / y20 / 3 = 6.666除算
割り算の整数部x // y20 // 3 = 6整数除算
割り算の剰余x % y20 % 3 = 2余り
負数-x-20 = 20負数若しくは反対値に変換
正数+x+20 = 20正数若しくは数値の符号を保持する
べき乗x ** y203 = 8000指数演算 xy

たとえば、次の加算、減算、乗算、除算を実行します。

15 + 7        # 22
3.4 + 2.225   # 5.625
7 + 4.5       # 11.5

異なる数値型間でのPythonの算術演算子使用

上記では、算術演算子を使用して同じ型の数値間を計算すると、返される結果は元の数値と同じデータ型の数値になることがわかっています。

算術演算子を使用して異なる型の数の間で計算する場合は異なり、返される結果は、各元のデータ型の中に一番制限が少ないデータ型になります。

Pythonでの数値データ型の低いものから高いものへの順序は次のとおりです。

複素数>実数>整数
