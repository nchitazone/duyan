Pythonの数値型

English Vietnamese

21 1月 ,2021

整数型（int)、浮動小数点型（float）、複素数（complex)の3種類をサポートしています。ここでは、これらのデータ型の使用方法について説明します。

Pythonの数値型とは

Pythonの数値型とはPythonプログラムで使用される数字を呼び出すために使用されるデータ型の名前です。文字リテラルと同様にPythonの数値データ型もリテラルです。これはそれ自体を表す値との意味です。

たとえば数値の11は数値データ型であり、それ自体の値は11を表し、外部要因に依存しません。

整数データ型(int)

**Pythonの整数データ型(int)**には、正の整数（1,2,3…）、負の整数（-1、-2、-3…）、および数値0が含まれます。整数自体にはコンマは含まれません。
