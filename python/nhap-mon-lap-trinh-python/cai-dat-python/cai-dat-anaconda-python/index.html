Cài đặt anaconda python


20 tháng 8 ,2020

Hướng dẫn cài đặt anaconda python. Bạn sẽ học được cách cài đặt anaconda python cho windows 7 hoặc 10, cũng như cách cài đặt anaconda cho macos sau bài học này.

Anaconda python là gì

Khái niệm anaconda python

Anaconda là một nền tảng phân phối ngôn ngữ lập trình python miễn phí, phục vụ cho tính toán khoa học và phát triển trí tuệ nhân tạo.
Anaconda bao gồm ngôn ngữ lập trình python cộng với các thư viện và gói package cần thiết thường dùng, với mục đích quản lý và triển khai môi trường lập trình python dễ dàng hơn.

Thông thường, để sử dụng Python, sau khi cài đặt Python, chúng ta cần phải cài đặt thêm các thư viện và các gói package cần thiết để chạy chương trình.
Ví dụ như để phát triển AI bằng python, chúng ta cần cài đặt thêm Tensorflow, matplotlib, Open CV chẳng hạn.
Tuy nhiên, nếu bạn cài đặt anaconda python, không những python mà các thư viện và gói package thường dùng cũng sẽ được cài đặt luôn theo đó.

Ưu nhược điểm của cài đặt anaconda python

So với việc chỉ cài đặt python, thì việc cài đặt anaconda python có ưu thế lớn nhất là, ngoài python ra thì có khoảng hơn 600 package mở rộng vốn không nằm trong bộ package chuẩn của python sẽ được cài đặt kèm, và bạn chắc chắn sẽ cài đặt được môi trường lập trình python sau khi cài đặt cài đặt anaconda python.

Ngược lại, với những lập trình viên đã có kinh nghiệm lập trình thì việc cài đặt anaconda python sẽ khiến họ khó tạo lập một môi trường lập trình python riêng biệt cho mình.

Đối với người mới bắt đầu học lập trình python, thì ưu điểm của cài đặt anaconda python có lẽ sẽ nhiều hơn là nhược điểm.

Cài đặt anaconda python cho windows

Tải anaconda python cho windows

Bạn hãy tải anaconda python bằng link gốc tại website chính thức của Anaconda https://www.anaconda.com

Chọn menu Products > Individual Edition, sau đó kéo xuống phía dưới và tìm đến phần Anaconda Installers như sau:

Hãy chọn phiên bản phù hợp với hệ điều hành máy tính của bạn bằng cách click và tải file về máy.

Cài đặt anaconda python cho windows

Tại thời điểm viết bài này thì phiên bản mới nhất của Anaconda cho windows là Anaconda3-2020.11-Windows-x86_64.exe.
Sau khi tải file này về máy, hãy nhấn đúp chuột vào file cài đặt anaconda python để khởi động trình cài đặt anaconda python.

Hãy nhấn vào nút NEXT>, rồi I Agree.

Khi đó sẽ hiện ra màn hình dưới đây. Nếu bạn muốn cài đặt anaconda python cho nhiều người dùng, hãy tích vào All users(requires admin privileges). Nếu bạn chỉ muốn cài đặt anaconda python chỉ cho riêng tài khoản của mình, hãy giữ nguyên chọn lựa Just Me (recommended) và nhấn vào nút NEXT>.

Sau đó hãy chọn vị trí cài anaconda python trong máy bằng cách nhấn vào nút Browse, hoặc giữ nguyên mặc định cài đặt và nhấn vào nút NEXT>.

Tại màn hình tiếp theo, nếu bạn muốn cài path anaconda python cho windows, hãy tích vào Add Anaconda3 to PATH enviroment variable. Sau khi hoàn tất cài đăt anaconda python cho windows, bạn có thể dùng Anaconda python thông qua màn hình DOS của máy tính.
Tuy nhiên với sự lựa chọn này sẽ gây một số khó khăn cho bạn khi phải xóa Anaconda và cài mới nó.
Bạn cũng có thể tích vào Register Anaconda3 as my default Python 3.7, để các phần mềm dùng cho Python như Visual Studio PyCharm, hay Wing IDE sẽ tự nhận diện Anaconda như là phiên bản chính của Python trên hệ điều hành.

Sau khi lựa chọn xong các option, hãy nhấn vào nút Install để bắt đầu quá trình cài đặt anaconda python cho windows.

Cuối cùng hãy nhấn nút Finish và kết thúc quá trình cài đặt anaconda python cho windows.

Cài đặt anaconda cho maxos

Cách cài đặt anaconda cho maxos cũng tương tự như khi bạn cài đặt anaconda python, hãy chọn phiên bản anaconda dành cho macos từ link tải trực tiếp trên website chính thức của anaconda https://www.anaconda.com/distribution/.

Sau đó hãy anaconda cho maxos và kiểm tra phiên bản tương tự như Windows mà Kiyoshi đã hướng dẫn ở trên.

Test phần mềm anaconda python mới cài đặt

Sau khi cài đặt Anaconda python thành công, hãy khởi động Anaconda Prompt. Bạn có thể tìm thấy chương trình trên thanh Taskbar hoặc nhấn phím Window + F và nhập tên Anaconda Prompt để tìm kiếm icon chương trình. Hãy click vào icon để khởi động Anaconda Prompt.

Sau đó, hãy thử kiểm tra phiên bản của Python và Anaconda bằng cách nhập dòng lệnh dưới đây và nhấn Enter.

Nếu thông tin version của python và anaconda hiện ra như dưới đây thì chúc mừng bạn đã cài đặt anaconda python thành công.
