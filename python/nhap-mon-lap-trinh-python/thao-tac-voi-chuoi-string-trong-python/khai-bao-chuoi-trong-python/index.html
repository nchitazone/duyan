Khai báo chuỗi trong python


16 tháng 2 ,2021

Hướng dẫn cách khai báo chuỗi trong python. Bạn sẽ học được cách khai báo chuỗi bằng dấu nháy đơn và nháy kép trong python, cũng như cách khai báo chuỗi nhiều dòng trong python sau bài học này.

Chuỗi trong python ( tiếng anh : String in python) là một hằng số bao gồm nhiều ký tự nối lại với nhau. Chuỗi ký tự cũng là một iterable - dạng dữ liệu là các đối tượng có nhiều phần tử có thể lặp lại.
